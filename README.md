# Буткемп для начинающих мобильных разработчиков

Найди своё течение в океане мобильной разработки.

Знакомство с актуальными технологиями нативной и кроссплатформенной мобильной разработки: Jetpack Compose (Android), Swift UI (iOS), Flutter и Kotlin Multiplatform.

Практические занятия, погружающие в процесс создания приложений: от инициализации Git-репозитория до описания с применением возможностей Github Actions действий по непрерывной интеграции (CI).
